# 00-inbox/

Default landing folder for brand-new notes.

**Rule of thumb:** write the note here while it's half-baked. Once it settles
into a category — architecture, runbook, playbook, reference, fix/audit —
move it out.

Obsidian is configured (or will be) to create new notes here by default:
*Settings → Files and links → Default location for new notes → `00-inbox/`*.

Nothing here should be linked from `INDEX.md` or `MOC.md` permanently —
if something lives in the inbox for more than a week, it needs a real home.
